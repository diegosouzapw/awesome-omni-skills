## Grounding Scorecard

| Dimension | Strong signal | Weak signal |
| :-------- | :------------ | :---------- |
| Retrieval recall | Relevant document appears in top results | Correct answer exists but never retrieves |
| Citation precision | Section-level reference is present | Citation points to a whole document vaguely |
| Freshness | Timestamp or freshness window is shown | User cannot tell if evidence is stale |
| Abstention | Weak evidence narrows claims | System overstates confidence |
