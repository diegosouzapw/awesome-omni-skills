# Risk Hotspots Checklist

Use this checklist before finalizing a PR body.

## Hotspot Scan

- Did the PR change runtime behavior?
- Did the PR change contracts, schemas, or config files?
- Did the PR add rollout or migration risk?
- Did the PR leave any known follow-up work out of scope?
