## Signup strong, activation weak

Inspect:

- onboarding friction
- empty states
- setup burden
- unclear first action

Likely fix:

- redesign path to first value

## Activation strong, monetization weak

Inspect:

- upgrade triggers
- packaging
- buyer/user mismatch

Likely fix:

- align paid plans to meaningful outcomes

## PQL volume high, quality low

Inspect:

- noisy events
- missing fit criteria
- no exclusions

Likely fix:

- tighten signal design and qualification window

## Trial weak, requests for extensions common

Inspect:

- trial length
- setup dependencies
- time needed for proof

Likely fix:

- change model or reduce setup to value

## One motion does not fit all

Inspect:

- segment differences
- stakeholder count
- procurement burden

Likely fix:

- split motion by segment
