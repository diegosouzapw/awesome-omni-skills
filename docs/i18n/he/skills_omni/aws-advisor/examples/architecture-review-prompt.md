```text
Use @aws-advisor to review this AWS workload.

Context:
- workload type:
- users / traffic:
- availability target:
- RTO / RPO:
- compliance needs:
- regions involved:
- budget sensitivity:
- preferred AWS services or constraints:
- current IaC / delivery tooling:

Please:
1. identify assumptions and missing information
2. review the design across security, reliability, operations, performance, and cost
3. compare at least one alternative approach when trade-offs are significant
4. recommend next validation steps before implementation
```
