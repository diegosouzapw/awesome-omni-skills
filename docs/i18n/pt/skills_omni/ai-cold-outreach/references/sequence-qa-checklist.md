Use this checklist before launch or when rewriting an underperforming sequence.

## 1. Targeting fit

- [ ] Sequence is matched to a defined segment
- [ ] The first line uses a real business signal, not a vague compliment
- [ ] The offer fits the recipient's likely role and timing

## 2. First-touch hygiene

- [ ] No unnecessary links in the first email
- [ ] No images or attachments in the first email
- [ ] One CTA only
- [ ] Sender identity is clear
- [ ] Subject line is simple and not gimmicky

## 3. Claim safety

- [ ] No unverifiable superlatives
- [ ] No fabricated social proof
- [ ] No intrusive personal inference
- [ ] No fake familiarity

## 4. Personalization quality

- [ ] Personalization is specific
- [ ] Personalization is grounded in known facts
- [ ] Personalization is not creepy or surveillance-heavy
- [ ] The rest of the email still makes sense if the first line is removed

## 5. Sequence structure

- [ ] Each step has one job
- [ ] Follow-ups add a new angle, not just “bumping this” repetition
- [ ] CTA friction is appropriate for cold outreach
- [ ] Sequence length matches the sales motion

## 6. Reply handling

- [ ] Positive replies have an owner
- [ ] Objections have a response plan
- [ ] Opt-outs are suppressed everywhere they need to be
- [ ] OOO replies have a pause / resume rule

## 7. Launch recommendation

Mark one:

- [ ] Go
- [ ] Revise before send
- [ ] Pause until infrastructure/compliance issue is fixed
