## Scenario

A developer tool has strong self-serve adoption, but larger accounts need security review, admin controls, and rollout support.

## Recommendation

- Segment SMB and mid-market as self-serve first
- Trigger sales only after fit + value + buying intent are visible
- Example PQL: ICP account, activation completed, 5+ active users from same domain, admin page visit, enterprise feature interest
- Sales role: support rollout, procurement, security, and expansion

## Why

- product proves value first
- sales adds help where product alone is insufficient
- avoids premature rep contact that damages PLG trust
