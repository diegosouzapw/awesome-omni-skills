Use this note when the founder's request starts inside solo-founder GTM but drifts into a narrower specialization.

## Route to Pricing

Route when the core issue is:

- packaging and plan design
- annual plans or discount logic
- willingness-to-pay testing
- monetization structure

## Route to Outreach / SDR

Route when the core issue is:

- sequence design
- outbound copy iteration
- prospecting operations
- reply handling workflows
- SDR handoff process

## Route to Analytics / Instrumentation

Route when the core issue is:

- event taxonomy design
- dashboard implementation
- analytics QA
- warehouse or tracking setup

## Route to Legal / Compliance Review

Route when the founder asks for:

- jurisdiction-specific legal interpretation
- consent or lawful-basis determinations
- definitive compliance answers
- review of sensitive data handling obligations

## Preserve Context During Handoff

When handing off, include:

- founder stage
- current MRR and ACV
- dominant motion
- current bottleneck
- key metrics already known
- constraints around privacy, tooling, or founder time
