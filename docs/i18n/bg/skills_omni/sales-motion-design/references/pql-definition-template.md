## Handoff timing

Sales should enter only when both are true:

1. The account has shown meaningful product value.
2. Sales can add something the product alone cannot.

## Sales action after handoff

Define what sales should actually do.

- Help with rollout planning
- Support procurement or security review
- Explain enterprise packaging or pricing path
- Coordinate multi-stakeholder evaluation
- Assist integration or migration planning
