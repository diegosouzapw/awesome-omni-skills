## User request

```text
Create a new architecture decision record in the TECH space called "ADR-001: Use Event-Driven Notifications".
```

## Suggested agent flow

1. Confirm the Confluence site.
2. Resolve the `TECH` space.
3. Check for an existing ADR with the same title.
4. Draft the page body in the format expected by the MCP tool.
5. Create the page.
6. Return the page title and page ID.

## Good confirmation output

```text
Created page "ADR-001: Use Event-Driven Notifications" in the TECH space. Returned page ID: 123456789.
```
