```text
Mission:
- [one sentence]

Reported symptom:
- [error, wrong behavior, failing path]

Success criteria:
- [what must be true after the fix]

Known constraints:
- [environment, permissions, time, scope]

Likely entrypoints to search:
- [error text]
- [endpoint or command]
- [feature flag or config key]
- [related test names]

Recon notes:
- [file → finding]
- [file → finding]

Plan:
1. [step] → files: [expected files] → verify: [evidence]
2. [step] → files: [expected files] → verify: [evidence]
3. [step] → files: [expected files] → verify: [evidence]

Risks:
- [risk]

Manual validation if needed:
1. [step]
2. [step]
3. [expected result]

Notebook update:
- [new note, updated note, or none and why]
```
