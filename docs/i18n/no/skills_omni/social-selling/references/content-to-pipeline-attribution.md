Treat content as part of the selling system only if it has a visible path to conversation and opportunity tracking.

## Touch taxonomy

### First touch

The first meaningful social interaction, for example:

- saw a post
- connected after profile visit
- replied to a comment
- replied to an inbound DM

### Engaged touch

A later interaction that shows increased interest:

- repeat engagement with posts
- DM response
- asset request
- continued conversation about the problem

### Conversion touch

The interaction that directly leads to a meeting or sales step:

- request for more details
- acceptance of a meeting invite
- referral to another stakeholder
- explicit buying discussion

## Required CRM fields

- account name
- contact name
- source channel
- source subtype
- first-touch date
- trigger event
- content topic or asset
- DM opened date
- reply status
- meeting booked date
- opportunity created date
- current stage
- owner

## CTA map by content type

| Content type | Primary CTA | Secondary CTA |
| --- | --- | --- |
| Problem insight post | comment with opinion or question | DM for framework or example |
| Case study snippet | ask for details | request similar use-case discussion |
| Tactical post | save or comment | DM for checklist or template |
| Founder perspective | reply with experience | connect for discussion |
| Event or trigger post | direct response | follow-up DM |

## Weekly review

Review once per week:

1. Which posts created qualified conversations?
2. Which CTAs produced replies, not just reactions?
3. Which themes produced meetings?
4. Which segments responded best?
5. Where did handoff to CRM fail?

## What to avoid

- counting likes as pipeline
- using vague source labels like “social” only
- failing to distinguish inbound from outbound or content-assisted touches
