## Define the value moment

Complete this sentence:

> A new user has reached first value when they have successfully __________________.

## Map the path

List each step from first interaction to first value.

1. Discovery or landing page action
2. Sign-up or access step
3. Initial setup
4. Data or integration step
5. First meaningful action
6. First visible outcome

## Friction log

For each step, note:

- required effort
- likely drop-off risk
- dependency on another person or system
- whether the step can be removed, simplified, or preloaded

## Activation design

- What activation event best represents meaningful progress?
- What secondary signals indicate a user is likely to monetize?
- What are the biggest blockers before activation?

## Improvement checklist

- Remove nonessential form fields
- Avoid empty first-run states
- Use templates, sample data, or prebuilt workflows
- Delay advanced configuration until after first value
- Guide the next action contextually
- Measure time to activation by segment and acquisition source

## Output

Summarize:

- current TTFV estimate
- target TTFV
- top 3 friction points
- fastest changes to test first
