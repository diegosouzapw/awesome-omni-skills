Use this skill for strategic GTM motion design.

Route away when the task becomes primarily:

- pricing design or monetization packaging
- outbound copywriting or sequence generation
- CRM workflow implementation
- analytics instrumentation or dashboard setup
- enterprise sales process management
- software implementation or architecture

When handing off, preserve:

- the segment definitions
- motion recommendation
- assumptions and confidence level
- TTFV definition
- PQL or proof-of-value logic already established
