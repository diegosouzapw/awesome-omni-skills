## Scenario

A product gets many signups but very few paid conversions.

## Diagnosis path

1. Check whether activation is actually happening
2. Map time to first value
3. Review whether free users reach a meaningful outcome
4. Inspect upgrade triggers and package boundaries
5. Evaluate whether sales touches are early or irrelevant

## Likely finding

The product is measuring signups, not value. Users hit an empty first-run state and never complete the activation event.

## Recommendation

- preload a template or sample environment
- reduce first-run setup
- redefine activation around the first successful outcome
- revisit trial or freemium model only after activation improves
