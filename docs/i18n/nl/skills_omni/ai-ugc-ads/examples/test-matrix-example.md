## Objective

Improve conversion-oriented TikTok creative for a hydration product.

## Hypothesis

A demo-first hook will outperform a spoken pain-point hook for cold traffic.

## Controlled test

| Variant | Hook | Body | CTA | Creator | Audience | Notes |
| --- | --- | --- | --- | --- | --- | --- |
| A | Spoken pain point | Same | Same | Same | Same | Baseline |
| B | Demo-first visual | Same | Same | Same | Same | Tests hook only |
| C | Curiosity opening | Same | Same | Same | Same | Tests hook only |

## Decision notes

- Keep audience, offer, landing page, and CTA stable.
- Do not change creator or placement mix during this test unless documented.
- If results are inconclusive, rerun with fewer simultaneous changes.
