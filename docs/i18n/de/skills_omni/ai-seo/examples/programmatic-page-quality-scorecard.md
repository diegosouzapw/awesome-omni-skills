Score each page from 0-2 in each category.

| Category | 0 | 1 | 2 |
| --- | --- | --- | --- |
| Unique value | token swap only | some variation | materially distinct and useful |
| Answer-first clarity | vague opening | partial answer | direct answer near top |
| Evidence support | unsupported | mixed | sourced or verified |
| Internal linking | orphaned | minimal links | linked from hubs and related pages |
| Indexability readiness | unclear | mostly ready | sitemap/canonical/linking ready |
| Schema fit | misused | uncertain | clearly matches visible content |

## Decision guide

- **10-12:** ready for pilot publish
- **7-9:** revise before publish
- **0-6:** do not scale this template
