Use these as compact response structures.

## Assumptions mode

```text
Steelmanned thesis:
...

Hidden assumptions:
1. ... [assumption-risk]
2. ... [assumption-risk]
3. ... [assumption-risk]

Most decision-critical assumption:
...

What would validate or break it:
...
```

## Evidence audit mode

```text
Steelmanned thesis:
...

Key claims and support:
1. Claim: ...
   Grade: Direct|Indirect|Anecdotal|Assertion|Missing
   Gap: ...
   Why it matters: ...

Overall confidence:
HIGH|MEDIUM|LOW|PIVOT

Next validation action:
...
```

## Pre-mortem mode

```text
Steelmanned thesis:
...

Failure narrative 1:
- Trigger:
- Failure chain:
- Detection signal:
- Blast radius:

Failure narrative 2:
...

Residual risk:
...
Confidence:
...
```

## Red-team mode

```text
Steelmanned thesis:
...

Abuse case 1:
- Actor:
- Goal:
- Asset:
- Entry point:
- Preconditions:
- Weakness:
- Impact:

Residual adversarial risk:
...
```

## Argue-the-other-side mode

```text
Steelmanned thesis:
...

Strongest opposing case:
...

Why it may fit better:
- ...
- ...
- ...

Trade-offs still unresolved:
...
```
