Use these as patterns, not rigid automation scripts.

## Warm variant

### Connection request

```text
Hi [Name] — your post on [topic] was useful, especially the point about [specific detail]. I work with [ICP/problem area] and would be glad to connect.
```

### First DM

```text
Thanks for connecting. Your point about [specific context] stood out. We keep seeing [relevant pattern] with [similar companies/team type]. Curious if that is showing up on your side too?
```

### Follow-up

```text
One small resource in case it is useful: [brief description, no hard sell]. If helpful, I can send the version we use with clients.
```

## Lukewarm variant

Use when there is a clear trigger but little prior interaction.

### Connection request

```text
Hi [Name] — noticed [role change / hiring signal / relevant company update]. I work on [problem area] with teams in a similar stage, so your updates caught my attention. Happy to connect.
```

### First DM

```text
Thanks for accepting. Noticed [specific trigger]. That usually creates pressure around [problem]. Are you focused on that already, or is something else higher priority right now?
```

## After non-response

- Do not send repeated reformulations of the same pitch.
- If there is no response after reasonable follow-up, stop.
- Return to light nurture through relevant engagement or future trigger-based outreach.

## Negative example to avoid

```text
Hey [Name], we help companies like yours grow faster. Open to a 15-minute call this week?
```

Why it fails:

- generic
- pitch-first
- no visible reason for contact
- too much ask too early
