## Case 1: Discovered, not indexed

Check in order:

1. Is the page in the sitemap?
2. Is the page linked internally?
3. Is the canonical correct?
4. Is the page too similar to other pages?
5. Is the template creating low-value variants?

## Case 2: Impressions up, clicks flat

Check in order:

1. Does the title reflect actual query intent?
2. Is the opening answer too weak or too generic?
3. Is the page differentiated from competing results?
4. Are you winning the wrong impressions?

## Case 3: Rich result loss after template update

Check in order:

1. Is the marked-up content still visible?
2. Did the page type change?
3. Is the schema still appropriate?
4. Was useful content removed from the page?
