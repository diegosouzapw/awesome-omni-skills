## 1. Growth feels slow. What is actually weak?

- If traffic is low -> review acquisition channel focus.
- If traffic is fine but activation is low -> fix onboarding and time to value.
- If demos happen but closes are low -> improve qualification and ROI framing.
- If customers buy but churn quickly -> inspect onboarding quality and product fit.
- If all of the above are acceptable but founder time is saturated -> automate repetitive execution or prepare a hire.

## 2. Is the founder using too many channels?

- Yes -> cut to one primary motion and one secondary experiment.
- No -> continue.

## 3. Is data good enough to make decisions?

- No -> instrument the minimum KPI set and clean CRM stages.
- Yes -> continue.

## 4. Is AI helping or adding noise?

- If outputs are generic or risky -> reduce autonomy, add rubric, review samples.
- If outputs save time with acceptable quality -> keep and document the workflow.

## 5. Is hiring actually the answer?

- If demand is not repeatable -> no, keep learning.
- If demand is repeatable but handoffs are undocumented -> document first.
- If demand is repeatable and founder remains the bottleneck -> consider contractor, then FTE.
