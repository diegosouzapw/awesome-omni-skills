Use this prompt to turn raw background information into a sales-ready LinkedIn profile draft.

```text
Rewrite this LinkedIn profile for social selling.

Audience:
- ICP: [describe buyer]
- Offer: [describe service/product/outcome]
- Trust assets: [case studies, testimonials, metrics, media]
- Preferred CTA: [connect / DM / book intro / request resource]

Current background:
[paste raw bio, resume bullets, current headline, About section, and proof points]

Output:
1. 3 headline options
2. 1 banner message suggestion
3. 1 About section using: problem -> method -> proof -> CTA
4. 3 Featured-section recommendations
5. 5 profile weaknesses to fix first

Constraints:
- Write for the buyer, not as a resume
- Keep claims specific and supportable
- Avoid hype, jargon, and generic growth language
```
