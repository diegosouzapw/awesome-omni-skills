Use this worksheet to compare tool classes and vendors without relying on stale pricing or benchmark claims.

## How to use it

Score each candidate as `High`, `Medium`, `Low`, or `Unknown`.

| Criterion | Why it matters | Candidate notes |
| --- | --- | --- |
| Best fit motion | Human-recorded, AI-personalized, interactive demo, or recap | |
| Target volume support | Confirms whether the workflow fits weekly/monthly send volume | |
| Trust and authenticity fit | Some segments respond better to clearly human video | |
| Likeness / avatar / voice controls | Important for synthetic-media governance | |
| Pronunciation handling | Critical for personalized names and companies | |
| Review and approval workflow | Needed before batch generation or launch | |
| Hosted page and thumbnail delivery | Email-safe delivery matters more than flashy embeds | |
| CRM or sequencer compatibility | Confirms whether engagement can be logged reliably | |
| Analytics export | Needed for reporting and comparison across tools | |
| Multilingual support | Important for international campaigns | |
| Interactive demo support | Needed when replacing intro meetings or supporting champions | |
| Team training burden | Some workflows are operationally heavier than they appear | |
| Data retention and admin controls | Important for screenshots, uploaded media, and generated assets | |
| Fallback path if blocked | Example: switch from synthetic to human-recorded or text-first | |
| Verify current plan details | Pricing, limits, and features drift frequently | |

## Practical selection guidance

### Human-recorded first
Prefer this when:
- ACV is high
- authenticity is critical
- volume is manageable
- post-demo recap or proposal walkthrough is the main use case

### Hybrid or AI-assisted human video
Prefer this when:
- volume is growing
- the team still wants a recognizably human feel
- moderate personalization is enough
- CRM logging and review workflows exist

### Synthetic personalization
Prefer this when:
- volume is high enough to justify operational setup
- the organization has clear policy on disclosure and authorization
- the message remains simple, honest, and tightly quality-checked

### Interactive demo-led motion
Prefer this when:
- the buyer needs to explore the product asynchronously
- the content must be shared internally
- product context matters more than presenter presence

## Verify before purchase or rollout

- trial and approval limits
- supported personalization fields
- CRM and sequencer integrations
- exportable analytics definitions
- language and pronunciation QA process
- admin controls and asset retention options
- whether the workflow still works if synthetic generation is disabled
